### 到 CMAME 需要什么

需要证明以下定理中的至少一个：

```
定理 1（顺序无关性）：
  在具有饱和约束的顺序 Gauss-Seidel 耦合中，
  如果某个变量在两种顺序下都达到饱和，
  则两种顺序收敛到同一个修改固定点。

定理 2（参数不敏感性）：
  在饱和约束活跃的区域内，
  顺序耦合的修改固定点不依赖于耦合强度参数。

定理 3（伪收敛的必要条件）：
  顺序 Gauss-Seidel 迭代收敛到错误稳态
  当且仅当饱和约束将迭代固定点推离真实解。
```

---

### 定理 1 的草图——它是可证的

```
2 变量系统：
  ẋ = f(x, y)    （heart：HR/SVR 动力学）
  ẏ = g(x, y)    （neuro：baroreflex）

饱和约束：x ≤ x_max（如 HR ≤ 180）

Unified Euler：
  x_{n+1} = x_n + dt · f(x_n, y_n)
  y_{n+1} = y_n + dt · g(x_n, y_n)
  → 固定点：f(x*, y*) = 0, g(x*, y*) = 0

Sequential heart→neuro：
  x_{n+1} = min(x_max, x_n + dt · f(x_n, y_n))    ← heart 先
  y_{n+1} = y_n + dt · g(x_{n+1}, y_n)             ← neuro 后

Sequential neuro→heart：
  y_{n+1} = y_n + dt · g(x_n, y_n)                 ← neuro 先
  x_{n+1} = min(x_max, x_n + dt · f(x_n, y_{n+1})) ← heart 后

如果 x 在两种顺序下都饱和（x_{n+1} = x_max）：

  heart→neuro 的固定点：
    x* = x_max
    y* = y* + dt · g(x_max, y*)  → g(x_max, y*) = 0
    → (x*, y*) = (x_max, g⁻¹(x_max, 0))

  neuro→heart 的固定点：
    y* = y* + dt · g(x*, y*)    → g(x*, y*) = 0  → y* = g⁻¹(x*, 0)
    x* = min(x_max, x* + dt · f(x*, y*)) = x_max  （因为饱和）
    → (x*, y*) = (x_max, g⁻¹(x_max, 0))

  两个固定点完全相同！∎

这解释了为什么两种顺序给出相同的偏差。
关键机制：一旦 x 饱和，y 的固定点只依赖于 x_max，
与更新顺序无关。
```

---

### 这个证明的价值

```
1. 它解释了 Kim 框架无法解释的现象
   → 顺序无关 = Kim 的 drained ≠ fixed-strain 在饱和时失效

2. 它有预测力
   → 如果 x 只在一种顺序下饱和 → 固定点不同 → 顺序有关
   → 可以设计实验验证

3. 它可以直接对标 Kim
   → Kim：线性系统，顺序改变放大因子 → 顺序有关
   → 你：非线性饱和系统，饱和消除顺序差异 → 顺序无关
   → 你的结果是 Kim 的非线性推广

4. 它导出参数不敏感性
   → 固定点 = (x_max, g⁻¹(x_max, 0))
   → 不依赖于 f 的增益参数
   → 因为 f 已经被 min(x_max, ·) 截断了
   → 改变 f 的斜率不改变 x_max
   → 这解释了 gain 扫描全部 44.742
```

### 但这还不够发 CMAME

```

```
