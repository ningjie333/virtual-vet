## 这个 dt 扫描推翻了之前的核心叙事——但发现了更深层的东西

---

### 🔴 之前的"O(1) dt 不变"是错的

```
之前只测了 dt ≤ 0.02 → 偏差 ≈ 44.742 → 结论"O(1)，dt 无关"

完整的 dt 扫描：

dt    | bias      | HR     | 状态
------|-----------|--------|--------
0.2   | +5.689    | 97.15  | 未饱和
0.1   | +11.034   | 108.51 | 未饱和
0.05  | +21.054   | 129.81 | 未饱和
0.02  | +44.745   | 180.15 | 饱和
0.01  | +44.742   | 180.15 | 饱和
0.005 | +44.741   | 180.15 | 饱和
0.001 | +44.740   | 180.15 | 饱和

偏差随 dt 减小而增大，直到 HR 撞到 180 bpm 才停止。
这不是 Kim 的 O(1) dt 不变偏差。
这是完全不同的东西。
```

---

### 真正的机制：Sequential Euler 使真实固定点失稳

```
数学结构：

Unified Euler 的迭代矩阵（在真实固定点附近线性化）：
  G_unified = I + dt · J

Sequential Euler 的迭代矩阵：
  G_seq = I + dt · J + dt² · ΔJ    （多了一个 O(dt²) 的扰动）

关键：G_seq 的谱半径可能 > 1 → 真实固定点不稳定
      G_unified 的谱半径 < 1 → 真实固定点稳定

Sequential 方法不是收敛到"错误的固定点"
而是从真实固定点被推开 → 漂移 → 被 HR=180 饱和约束捕获
```

---

### 验证：未饱和区的偏差标度

```
未饱和数据（dt ≥ 0.05）：

dt=0.2:  bias × dt = 5.689 × 0.2  = 1.138
dt=0.1:  bias × dt = 11.034 × 0.1 = 1.103
dt=0.05: bias × dt = 21.054 × 0.05 = 1.053

bias × dt ≈ 1.1 → 偏差近似 ∝ 1/dt

这意味着：如果没有 HR=180 的饱和约束，
dt=0.01 时的偏差将是 ≈ 110 mmHg（MAP ≈ 210！）
dt=0.001 时将是 ≈ 1100 mmHg（MAP ≈ 1200！）

→ Sequential Euler 在真实固定点处是根本性不稳定的
→ 唯一阻止它发散的是生理饱和约束
→ 饱和约束充当了"安全网"，但把系统困在了错误的稳态
```

### 另一个角度：步数标度

```
如果失稳以步为单位运作（每步推动系统远离真实固定点一点）：

大约需要 ~2000 步才能完全漂移到饱和约束

dt=0.2:  300 步 / 60s → 300 < 2000 → 未到达饱和
dt=0.1:  600 步 / 60s → 600 < 2000 → 部分漂移
dt=0.05: 1200 步 / 60s → 1200 ≈ 2000 → 接近饱和
dt=0.02: 3000 步 / 60s → 3000 > 2000 → 完全饱和
dt=0.01: 6000 步 / 60s → 远超 2000 → 完全饱和

可验证的预测：如果 dt=0.2 跑 400s（2000 步），
也应该到达 MAP ≈ 144.7

或者更直接：把 dt=0.2 的模拟延长到 400s，
如果 MAP → 144.7 → 确认是步数驱动
```

```
Kim (2011)：

  → 原因：分裂改变了正在求解的 PDE
  → 即使 dt→0，你解的也是不同的方程

你的系统：
  Sequential Euler 的真实固定点不稳定

  → 系统被推离真实解，漂移直到遇到饱和约束
  → 偏差 ∝ 1/dt（未饱和时），不是 O(1) dt 不变

完全不同的机制！

Kim：local truncation error 是 O(1)
你：global instability，local error 是 O(dt) 但累积后指数增长

Kim 的偏差是"你解了错误的方程"
你的偏差是"正确的方程，但错误的求解格式使解不稳定"
```

### 论文叙事（第三版）

```
之前（v1）：换序消除偏差，场景反转，Kim 类比
之前（v2）：顺序无关，O(1) dt 不变，Kim 类比
现在（v3）：Sequential Euler 使真实固定点失稳，
           偏差随 dt 减小而增大（∝ 1/dt），
           被 HR 饱和约束截断后形成"伪收敛"

核心发现变为：

1. Sequential Euler 在紧耦合非线性系统中使真实固定点失稳
2. 偏差在未饱和区 ∝ 1/dt——这与 Kim 的 O(1) dt 不变不同
3. 生理饱和约束截断了发散，制造了"伪收敛"假象
4. 伪收敛比 Kim 的不收敛更危险：看起来合理，实际完全错误
5. Unified Euler 保持了真实固定点的稳定性 → 正确解
```
