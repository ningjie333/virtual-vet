## 论文大纲（定稿版）

---

### 题目

**Numerical consequences of operator splitting in modular zero-dimensional cardiovascular models for hemorrhagic shock simulation**

### 核心论点

In modular zero-dimensional cardiovascular models, sequential coupling may introduce operator-splitting errors comparable to clinically relevant pressure differences during acute hemodynamic transitions. Internal coupling strategies should therefore be explicitly reported and verified as part of numerical model credibility assessment.

---

### Abstract（~250 词）

```
Lumped-parameter cardiovascular models are widely used for
simulating hemodynamic responses, yet the choice of numerical
coupling strategy between submodules remains largely overlooked.
This paper compares three time integration approaches for a
45-variable canine hemorrhagic shock model: (1) forward Euler
on the unified RHS, (2) sequential (partitioned) Euler with
operator splitting, and (3) implicit Radau IIA on the unified RHS
as reference. We show that while the sequential approach
achieves numerical stability via Gauss-Seidel-type ordering,
it introduces a step-size-independent splitting error—
converging to a different steady state than the reference
solution (MAP = 111.5 vs 98.1 mmHg, Δ = 13.5 mmHg). Unlike
truncation error, this bias cannot be reduced by decreasing
the step size (log-log slope = -0.21 for RMSE vs dt).
Stiffness analysis confirms that the system stiffness ratio
increases significantly under severe hemorrhage (njev/step
×11.4 at 800 mL), further motivating the use of implicit
solvers. In contrast, unified Euler achieves RMSE ~0.11 mmHg
at dt=0.01, demonstrating that stiffness itself is not the
primary limiting factor—sequential coupling is.
These findings suggest that sequential coupling in
modular zero-dimensional cardiovascular models may introduce
operator-splitting errors comparable to clinically relevant
pressure differences during acute hemodynamic transitions.
Internal coupling strategies should therefore be explicitly
reported and verified as part of numerical model credibility
assessment.
```

#### 2.2 Time Integration Strategies

**三种策略的数学表述——论文的方法核心。**

```
2.2.1  统一 RHS + 显式 Euler（Pure Euler）

       dy/dt = f(y),   y_{n+1} = y_n + Δt · f(y_n)

       - f(y) 是统一的 45 变量右端函数，所有模块对同一个 y 求导
       - 一阶精度，一阶一致（consistent）
       - 稳定性要求 Δt < Δt_crit = 2/|λ_max|
       - 分裂误差：无

2.2.2  顺序耦合 + 显式 Euler（Sequential Euler）

       Lie-Trotter splitting:
         y_A^{n+1} = y_A^n + Δt · f_A(y_A^n, y_B^n)      ← 用旧 B
         y_B^{n+1} = y_B^n + Δt · f_B(y_A^{n+1}, y_B^n)   ← 用新 A

       - Gauss-Seidel 型顺序更新
       - 一阶精度，但分裂误差常数 C_split 可能远大于 1
       - 分裂误差：ε_split = C_split · Δt + O(Δt²)
         其中 C_split 取决于耦合强度和子系统刚性

2.2.3  统一 RHS + 隐式 Radau IIA（Reference）

       Radau IIA (5th order, L-stable)
       - 自适应步长 + 自适应阶
       - 统一求解所有 45 变量
       - 作为 reference solution（rtol=1e-10）
       - 交叉验证：BDF(rtol=1e-8) 和 DOP853(rtol=1e-10)
         在 MAP 上差异 < 0.01 mmHg
```

**Figure 1**：三种策略的信息流示意图

```
(a) Pure Euler           (b) Sequential           (c) Radau IIA
    ┌───┐                 ┌───┐ y_A^{n+1}        ┌───────────┐
    │ f │──→ y^{n+1}      │ A │───────→┌───┐    │ Unified   │
    │   │                 │   │        │ B │    │ f(y)      │
    └───┘                 └───┘  ────→ └───┘    │ → Radau   │
                          y_B^n     y_B^{n+1}   │ → y^{n+1} │
    全部同步更新          A→B 顺序，B→A 延迟    └───────────┘
                                             全部联立求解
```

#### 2.5 Simplified Two-Variable Analysis

```
- 从 45 变量模型中提取 baroreflex 核心反馈环：
  dMAP/dt = (HR·SV - MAP/SVR) / C
  dHR/dt  = α · (MAP_set - MAP)

- 对 Lie splitting 做局部截断误差推导：
  ε_local ≈ (α · Δt / 2) · |dMAP/dt|

- 量级估计：
  α ≈ 0.3 s⁻¹（压力感受器增益）
  Δt = 0.01s, |dMAP/dt| ≈ 40 mmHg/s（失血时）
  ε_local ≈ 0.06 mmHg/step
  → 600 步累积 ≈ 36 mmHg（与观测值 ~30 mmHg 量级一致）

- 关键预测：
  误差 ∝ α · Δt · |dMAP/dt|
  → α 大（强反馈）→ 误差大
  → |dMAP/dt| 大（快速失血）→ 误差大
  → 两者乘积在正常稳态小，在休克时大
  → 解释了为何误差随休克强度非线性增长
```

#### 3.3 Splitting Error in Sequential Euler

Figure 3a shows RMSE versus step size for Pure Euler and
Sequential Euler on a log-log scale. Pure Euler converges
to a plateau of ~0.11 mmHg for dt ≤ 0.1s, well below
clinical relevance. In striking contrast, Sequential Euler
exhibits a nearly flat RMSE profile at ~11 mmHg, with a
log-log slope of -0.21—indicating that the error is
essentially independent of step size.

Crucially, the steady-state MAP under sequential coupling
stabilizes at 111.5 mmHg regardless of dt, whereas the
BDF reference gives 98.1 mmHg. This 13.5 mmHg offset is
not a truncation artifact: it persists at all step sizes
and reflects a systematic shift in the equilibrium point
caused by Gauss-Seidel-type information lag in the
baroreflex feedback loop.

We interpret the slightly negative slope (-0.21) as follows:
at larger dt, the transient is under-resolved and the
system has not fully converged to its (incorrect) steady
state, yielding a deceptively lower RMSE. As dt decreases,
the system more fully settles into the wrong equilibrium,
and RMSE approaches its true systematic-bias value of
~11-12 mmHg.

### 4. Discussion

#### 4.1 Why Sequential Coupling Converges to a Different Steady State

The observed step-size-independent bias can be understood
through the structure of the baroreflex feedback loop.
Consider the simplified two-variable system:

  dMAP/dt = (HR·SV - MAP/SVR) / C
  dHR/dt  = (HR_target(MAP) - HR) / τ_symp

Under monolithic integration, the steady state satisfies:
  HR* = HR_target(MAP*)    and    HR*·SV = MAP*/SVR
  → a unique equilibrium MAP* = 98.1 mmHg

Under Gauss-Seidel sequential coupling, the baroreflex
module updates HR using the already-updated MAP value
from the same step. This introduces a systematic
directional bias: HR_target is evaluated at a MAP that
has already been modified by the hemodynamic module's
response to the previous HR. The resulting effective
equilibrium shifts to MAP* = 111.5 mmHg.

This is qualitatively different from truncation error:
- Truncation error is O(Δt) and vanishes as Δt → 0
- This splitting bias is O(1) and persists regardless of Δt
- It cannot be remedied by computational refinement
- It can only be eliminated by changing the coupling strategy

#### 4.2 Mechanism of Sequential Overcompensation

```
- Sequential MAP=111.5 的完整因果链：

  失血 → MAP↓ → baroreflex 用旧 MAP 计算 error
        → error 偏大（因为 MAP 已在 heart 步更新后更低）
        → sympathetic 激活过度
        → HR overshoot → CO overshoot → MAP 反弹至 111.5

  本质：Gauss-Seidel 顺序的信息延迟 → 正反馈放大

  与 Pure Euler 对比：
  统一 RHS 中 MAP 和 HR 同步更新 → 无信息延迟 → 无过补偿
```

#### 4.3 Clinically Relevant or Numerically Negligible?

```
- Sequential 的 MAP=111.5 vs 正确值 98.1
  → 偏差 13.5 mmHg
  → 在失血性休克分级中，这可能改变分级判断

- "稳定但有偏"比"不稳定"更危险：
  → 不稳定 → 用户会发现（数值爆炸）
  → 稳定但有偏 → 用户可能误信

- 误差随休克强度非线性增长的含义：
  → 模型在正常状态下校准通过，不代表在极端场景下可信

- 谨慎声明：
  → 本文评估的是同一模型在不同数值策略下的差异
  → 模型本身的生理真实性需要独立实验验证
  → "clinically relevant"指误差量级与临床决策阈值可比拟，
     不等同于"改变了真实患者的治疗结果"
```

#### 4.4 Picard Iteration as Theoretical Compromise

```
- 模块化 vs 精度的权衡是工程现实：
  → 统一 RHS 要求所有模块写进同一个方程组
  → 模块化支持独立开发、替换、测试

- 理论上，Picard 子迭代（Gauss-Seidel on modules）
  在每步内重复 k 次模块更新：
    y_A^{(j)} = y_n + dt · f_A(y_A^{(j-1)}, y_B^{(j-1)})
    y_B^{(j)} = y_n + dt · f_B(y_A^{(j)}, y_B^{(j-1)})
  k→∞ 时理论上收敛到统一 RHS 解（若迭代收敛）
  → 代价：每步增加 k× 模块计算

- 实际障碍：当前代码中模块间通过直接属性修改
  耦合（而非通过 y 向量），无法直接实现模块级 Picard
  → 这是代码架构限制，不是方法本身的限制

- 决策指南（理论分析，待实验验证）：
  │ 场景                 │ 推荐方法              │
  │─────────────────────│──────────────────────│
  │ 正常稳态仿真         │ Sequential (k=1)      │
  │ 中度病理状态         │ 待实现: Picard k=2-3  │
  │ 急性休克/闭环控制    │ 统一 RHS + 隐式       │
  │ 模型验证/基准测试    │ 统一 RHS + 隐式       │
```

#### 4.5 Relationship Between Stiffness and Splitting Error

```
- 刚性和分裂误差常数是两个独立的问题：
  - 刚性 → 显式方法需要极小 dt（Pure Euler 的问题）
  - 大分裂误差常数 → 顺序耦合在实用 dt 下不可靠（Sequential 的问题）

- 两者共同指向统一 RHS + 隐式求解，但原因不同
- 不应混淆：修复刚性不等同于修复分裂误差
- 统一 RHS + 显式 Euler 误差仅 ~0.1 mmHg 的事实证明：
  刚性本身不是主要问题，分裂误差才是
```

#### 4.6 Limitations

```
1. 单一物种（犬类），未验证人类模型
2. 0D 模型简化，未涉及 3D-0D 耦合
3. Sequential Euler 的 Gauss-Seidel 效应方向依赖
   （交换模块顺序可能得到不同的有偏结果）
4. dt_crit 的精确值未计算（需 Jacobian 特征值分析）
5. Picard 子迭代仅做理论讨论，代码架构不支持模块级 GS
   实现（模块间通过属性修改而非 y 向量传递）
6. "Clinically relevant"指误差量级与阈值可比拟，
   不等同于改变真实患者结局
```

```

3. 模块顺序对分裂误差方向的影响
```

### 5. Conclusion

```
Sequential coupling in modular zero-dimensional cardiovascular
models may introduce operator-splitting errors comparable to
clinically relevant pressure differences during acute hemodynamic
transitions. The error arises from information lag in Gauss-Seidel-
type ordering of strongly coupled feedback loops, particularly the
baroreflex, and is amplified under acute hemorrhagic conditions.
Unified explicit Euler achieves RMSE ~0.11 mmHg at dt=0.01,
demonstrating that the problem is not stiffness itself but
sequential coupling—stiffness can be managed with small dt,
but splitting error is structural. For scenarios requiring high
transient accuracy, unified RHS with implicit time integration
remains the most reliable approach. Internal coupling strategies
should therefore be explicitly reported and verified as part of
numerical model credibility assessment.
```
