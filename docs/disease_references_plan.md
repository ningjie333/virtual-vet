# Disease References Plan

## 目标
为每个疾病的诊断标准、关键症状、治疗方案添加文献支持

|------|--------|----------|
| pneumonia | P0 | 犬细菌性肺炎诊断指南 |
| acute_renal_failure | P0 | AKI分期标准（IRIS） |
| dilated_cardiomyopathy | P0 | DCM诊断标准（ACVIM） |
| diabetic_ketoacidosis | P0 | DKA管理共识 |
| pericardial_effusion | P1 | 心包积液诊断 |
| disseminated_intravascular_coagulation | P1 | DIC诊断标准 |

## 文献来源优先级
1. ACVIM共识声明/指南
2. EVECCS/IVECCS指南
3. Textbook of Veterinary Internal Medicine (Ettinger)
4. Veterinary Emergency and Critical Care (Silverstein)
5. JVECCS/JAVMA相关研究
6. 兽医内科学教材

## 输出格式
每个疾病将包含：
- 诊断标准引用
- 关键症状的病理生理依据
- 治疗方案的循证支持
- 预后数据
